# Веб-сервис для шифрования английских слов и текстов.  
Он позволяет быстро и удобно зашифровать данные с помощью различных алгоритмов.

## Как это работает

Процесс шифрования состоит из нескольких простых шагов:

1. **Выбор источника**
   - Ввести текст вручную  
   - Или загрузить файл

2. **Выбор шифра**
   - Выбрать нужный алгоритм шифрования

3. **Указание ключа**
   - Ввести ключ для шифрования

4. **Получение результата**
   - Мгновенный зашифрованный текст
